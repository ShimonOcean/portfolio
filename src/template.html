Face Me! (WIP)

Face Me! takes links of images and identifies a face in them using clarifai's API by putting a bounding box around the face it detects. Utilizes Express to handle server side and Postgres to store user information safely by using bcrypt for encryption and knex to handle database interaction.

forkify

forkify gives users recipes for their favorite dishes by using the food2fork API. Built using vanilla JS, the project also took advantage of the Model View Controller architecture in order to handle searching, liking, and viewing recipes.

Fault Tolerant Key/Value Store

Working with three others, we built a key/value store that handles get, put, and delete requests and keeps track of the number of key/value pairs stored. Utilized gossiping in order to keep the store causally consistent and implemented parition tolerance through the replication of shards. Built in Python & Flask, deployed in Docker.
